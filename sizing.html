CSS Sizing Units

Viewport Units (vw,vh)

50vw width (50% of screen width)

15vh height (15% of screen height)

vw = viewport width (1vw = 1% of screen width) vh = viewport height (1vh = 1% of screen height) These change when you resize your browser window!

REM units

This is 1rem (base font size)

This is 2rem (twice the base)

This is 0.8rem (smaller than base) rem = relative to root element's font size Usually 1rem = 16px, but it scales with user's browser settings

EM Units

Parent element (20px font-size)

Child element (1.5em = 30px)

em = relative to parent element's font size In the example: parent is 20px, child is 1.5em = 1.5 Ã— 20px = 30px

Percentage (%)

50% width of parent
75% width of parent
% = relative to parent element's size Width % is based on parent's width, height % on parent's height

Pixel Units (px)

100px 200px 300px
px = absolute pixel units These stay the same size no matter what - fixed and precise!
